# Evert

Evert is a strict-by-default functional systems language whose central design
rule is that power should be local, explicit, and non-contagious. The language
design keeps purity, laziness, algebraic data types, monads, algebraic effects,
local mutation, structured concurrency, ownership, capabilities, and unsafe
system work available without making any one mechanism the ambient shape of the
whole programme.

This repository is currently in the design and compiler-foundation stage. The
Rust package is still starter scaffolding; the normative work lives in the
documentation set.
